Let's define a hash-map as we seen earlier:
(def h {:foo "buu" :bar "bazz"})

Now try some of this magic:
(:bar h "not found")

This is actually a convenient version of (get h :bar "not found") and it says:

"Give me the value associated with the key :bar if it exists in the map, or else the string "not found"".

Try (:zar h "not found") to continue.
